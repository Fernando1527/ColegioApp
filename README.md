# Sistema de Gestión Escolar – ColegioApp

## Descripción

ColegioApp es una aplicación web desarrollada en ASP.NET Core MVC que permite gestionar información académica de alumnos, materias y sus calificaciones mediante expedientes.

---

## Funcionalidades Principales

* **Gestión de Alumnos:** Crear, editar, eliminar y consultar alumnos.
* **Gestión de Materias:** Administración completa de materias.
* **Expedientes Académicos:** Relación entre alumnos y materias, con registro de notas y observaciones.
* **Cálculo de Promedios:** Obtención automática del promedio de notas por alumno.
* **Visualización de Datos:** Representación gráfica de los promedios mediante gráficos de barras.

---

## Tecnologías Utilizadas

* ASP.NET Core MVC
* Entity Framework Core
* SQL Server / LocalDB
* Bootstrap
* Chart.js

---

## Estructura

El sistema sigue el patrón MVC:

* **Models:** Entidades (Alumno, Materia, Expediente)
* **Views:** Interfaces de usuario
* **Controllers:** Lógica del sistema
* **Data:** Configuración de base de datos

---

## Conclusión

El sistema permite administrar y analizar información académica de manera eficiente, integrando operaciones CRUD, relaciones entre entidades y visualización de datos.
